01

계정 맵핑

타사에서 운영 중인 광고계정을 이관받는 경우, 결제수단 변경에 앞서 해당 광고계정을 나스미디어 MCC에 먼저 맵핑 받아야 합니다.

계정맵핑 프로세스

맵핑 연결 초대를 보내고, 초대 수락(=계정 맵핑 수락)을 하게 되면 바로 적용됩니다.
STEP 1 계정을 받는 주체
  1. 맵핑 받을 MCC 로그인
  2. [계정] 실적 + 버튼 클릭 후 [기존 계정에 연결]
  3. 이관 받을 CID 입력 → 담당 관리자 지정
  4. [요청 보내기] 클릭
STEP 2 계정을 주는 주체
  1. 이관 예정 구글애즈 계정 로그인
  2. 관리 → 액세스 및 보안
  3. [관리자] 탭 → 링크요청 → 작업: 동의 → 엑세스 권한 부여
필수 확인 나스미디어 MCC로 계정 맵핑 완료 후, 관리 → 액세스 및 보안 → [관리자] 탭에서 소유자 권한이 '예'로 설정되어 있는지 반드시 확인해주세요.

타사(대행사·렙) 인보이스를, 나스미디어 인보이스로 전환하는 절차입니다.
아래 화면 예시를 참고하여 실제 Google Ads 화면에서 동일하게 진행해 주세요.

📍진행 위치  외부 지원 신청서가 아니라, Google Ads 계정 내 [결제 → 결제 정보 변경 내역]에서 직접 진행합니다.
STEP 1. 계정 로그인
나스미디어로 이관할 Google Ads 계정에 로그인합니다.
STEP 2. 지불자 변경 메뉴 진입
왼쪽 메뉴 [결제 → 결제 정보 변경 내역]으로 이동한 뒤, 화면 상단의 파란색 연필 아이콘을 클릭하세요.
STEP 3. 지불자 변경 선택
연필 아이콘을 클릭하면 나타나는 메뉴에서 [지불자 변경]을 선택하세요.
STEP 4. 결제 책임자 이전 유형 선택
같은 결제 프로필 내부 이동이 아니라 타사 → 나스미디어처럼 조직이 바뀌는 경우이므로 외부 결제 책임자 이전을 선택 후 "다음"을 클릭하세요.
