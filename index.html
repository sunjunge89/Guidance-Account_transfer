01

계정 맵핑

타사에서 운영 중인 광고계정을 이관받는 경우,
결제수단 변경에 앞서 해당 광고계정을 나스미디어 MCC에 먼저 맵핑 받아야 합니다.

계정맵핑 프로세스

맵핑 연결 초대를 보내고, 초대 수락(=계정 맵핑 수락)을 하게 되면 바로 적용됩니다.
STEP 1 계정을 받는 주체
  1. 맵핑 받을 MCC 로그인
  2. [계정] 실적 + 버튼 클릭 후 [기존 계정에 연결]
  3. 이관 받을 CID 입력 → 담당 관리자 지정
  4. [요청 보내기] 클릭
STEP 2 계정을 주는 주체
  1. 이관 예정 구글애즈 계정 로그인
  2. 관리 → 액세스 및 보안
  3. [관리자] 탭 → 링크요청 → 작업: 동의 → 엑세스 권한 부여
필수 확인 나스미디어 MCC로 계정 맵핑 완료 후, 관리 → 액세스 및 보안 → [관리자] 탭에서 소유자 권한이 '예'로 설정되어 있는지 반드시 확인해주세요.
02

직광고주 이관

광고주가 Google에 직접 결제하던 계정을,
나스미디어 인보이스(월별 청구) 결제 방식으로 전환하는 절차입니다.

사전 참고사항

신청 전 아래 사항을 먼저 확인합니다.
  • ✔️ 광고주와 결제방식 변경 일정 협의
  • ✔️ 현재 결제 프로필 담당자의 승인 가능 여부 확인
  • ✔️ 이관 대상 Google Ads CID 확인
  • ✔️ 계정을 연결할 MCC 확인
  • ✔️ 사용할 인보이스(결제 계정) 확인
※ 참고
결제방식 변경 신청 후에는 현재 결제 프로필 담당자와 신규 결제 프로필 담당자 모두 Google 승인 메일을 통해 승인 절차를 완료해야 최종 전환됩니다.

STEP 1. Google 신청서 작성

결제방식 변경 신청 Google 신청서 바로가기 →
FORM Google 신청서 입력 가이드
  1. 위 Google 신청서 바로가기로 접속 (하위 계정 또는 MCC 관리자 계정으로 로그인)
  2. 결제 책임자 이전(지불자 변경) 유형 문항 → 수동 또는 자동 결제에서 월별 인보이스로 전환 선택
  3. 이용 한도액 보유 여부 문항 → 사용할 한도액을 이미 부여받았다면 예, 새로 신청해야 한다면 아니요 선택
  4. 이관 대상 CID, 연결할 MCC, 사용할 인보이스 정보 입력 후 제출
필수 확인 이 요청은 하위 계정 또는 MCC의 관리자만 제출할 수 있습니다. 회사 결제 프로필에 속한 계정이라면, 하위 계정 또는 MCC의 관리자에게 연락하여 이 양식을 대신 제출하도록 요청하시기 바랍니다.
