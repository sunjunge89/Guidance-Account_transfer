계정 맵핑

구글계정 이관 시, 사전 진행 필요사항

광고주 혹은 타대행사에서 운영하던 계정을 나스미디어로 이관하는 경우,
사전에 계정 맵핑이 진행 되어야 합니다.

계정맵핑 프로세스

소요시간
맵핑 연결 초대를 보내고, 초대 수락(=계정 맵핑 수락)을 하게 되면 바로 적용됩니다.
프로세스

계정을 받는 사람이 진행합니다.

맵핑 원하는 MCC 로그인 › [계정] 실적 + 버튼 클릭 후
[기존 계정에 연결] › 이관 할 CID 입력 후
[요청 보내기] 클릭
스크린샷 영역
계정 맵핑 수락

계정을 주는 사람이 승인합니다.

이관 예정 구글애즈 계정 로그인 › 도구 및 설정 ›
[관리] 액세스 및 보안 › [관리자] 탭 내
맵핑 요청사항 승인 클릭
스크린샷 영역
계정맵핑 수락의 경우, 기존 운영주체였던 광고주 OR 기존 대행사가 처리 진행해주시면 됩니다.
🔴 계정 맵핑 진행 시, 나스미디어 매출로 집계되기 위해서 계정 소유자 권한 '예'로 설정할 수 있도록 안내 해주시기 바랍니다.
* 계정 소유자 권한 설정의 경우, [관리자] 탭 내 맵핑 이후 맵핑 된 나스미디어 MCC 소유자 탭 부분에 '예'로 설정 되어 있는지 확인해주시면 됩니다.
